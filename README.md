# Student Z-Score Calculator 📊

This project uses Python to process student exam marks and calculate their Z-scores. It is designed to handle real-world data issues like missing values.

## ✨ Features
- **Data Cleaning**: Automatically removes "AB" (Absent) entries.
- **Statistical Analysis**: Calculates Mean and Standard Deviation using `NumPy`.
- **CSV Integration**: Reads from and writes directly to `Student.csv` using `Pandas`.

## 🛠️ Technologies Used
- Python 3.x
- Pandas Library
- NumPy Library

## 🚀 How it Works
The script follows the standard Z-score formula:
$Z = (x - μ) / σ$
